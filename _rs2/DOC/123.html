개발 설명

레온 황제 시절부터 무기와 방어구의 개발이 가능하다. 아바론 성 외부의 동쪽 탑에는 대장간이 있다. 대장장이와 대화하면 "새로운 ○○○를 만들고 싶습니다."라고 말한다. 직접 종류를 선택할 수 없지만 1회 전투 후 다시 말을 걸면 순서가 변화한다.

발주 후 몬스터와 5회 인카운트하면 개발이 완료된다. (1회마다 대장장이의 대사가 변화) 개발이 완료된 장비품은 다음으로 연대 점프를 하면 상점에서 구입할 수 있다.
장비의 개발 비용은 개발 비용에 따라 3단계로 구별한다. (40만, 80만, 120만 크라운)

단, 개발하고 싶은 계통의 장비를 취급하는 상점이 제국령으로 있지 않으면 개발이 불가능하다. 해당 계통을 개발하기 위해 그 지역을 제국의 영토로 만들어야 한다.(대표적으로 창, 도끼)

개발 상세

개발 의뢰 → 프로토 타입 완성 → 양산화
의 순서로 진행된다.
  • 개발 의뢰부터 프로토 타입의 완성까지는 기본적으로 5회의 전투 횟수가 필요하다. (퇴각, 패배도 유효)
  • 5회를 채우지 않고, 도중에 연대 점프가 발생했을 경우 완성이 된다.
  • 프로토 타입 완성으로부터 양산화까지는 반드시 연대 점프가 필요. (1~2년 점프로는 불가)
  • 최종 황제 시대에 개발을 하면 양산화가 불가능하다. 따라서 무기/방어구 개발은 최종 황제가 등장하기 이전에 완료해야 한다.
  • 양산화 단계에서는 신규 가입 캐릭터의 초기 장비로 해당 무기를 소지하고 있으며, 상점에서 판매를 하고 있다.

개발 순서 변화

전투 1회마다 대장장이가 개발하고 싶다는 장비품의 종류가 변경된다.
대장장이 순서
무기(노인) 검 → 대검 → 도끼 → 곤봉 → 창 → 소검 → 활 → 도끼창 → 검 → (반복)
방어구(청년) 투구 → 모자 → (개발 불가) → 중갑 → 경갑 → 팔뚝 → 구족 → 전투복 → 방패 → 가다 → 투구 → (반복)
현 세대에서 개발 완료된 계통은 개발 불가 메시지가 뜬다. 순서는 동일
